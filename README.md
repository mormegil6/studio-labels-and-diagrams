# XLR Cable Color-Coding Labels Reference
XLR connector color-coding chart for quick visual identification of balanced audio cable lengths in professional studio and field recording environments.

## Purpose
These reference files provide a visual guide to the color-coding system for XLR cables. These labels are designed to work with the [3D-printed XLR cable label rings](https://www.thingiverse.com/thing:4136093) available on Thingiverse.

## Files
- **`xlr-labels.svg`** — Vector format labels (editable in Inkscape, Adobe Illustrator, etc.)
- **`xlr-labels.pdf`** — Ready-to-print PDF format
